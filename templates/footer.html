Fonte dos Dados

Os dados apresentados neste site são retirados da Secretaria de Segurança Pública de São Paulo (SSP).

Aviso Importante

Em caso de roubo ou assalto, procure imediatamente as autoridades competentes para registrar um boletim de ocorrência. Isso é essencial para que providências sejam tomadas e para que possamos manter nossa base de dados atualizada e precisa.

Colabore com a segurança pública e ajude a melhorar nossa base de dados!
